# UI Implementation Review Checklist

Use this checklist after the interface exists and before proposing broader redesign work.

## Icons And Visuals

- Use real SVG icon sets instead of emoji-as-icon shortcuts
- Keep icon sizing and stroke style consistent
- Verify brand marks against official assets instead of approximating them

## Interaction Quality

- Interactive cards and controls show clear hover or focus feedback
- Clickable surfaces use pointer affordance where appropriate
- Transitions support clarity instead of causing layout shift

## Contrast And Legibility

- Light-mode surfaces keep enough opacity and border contrast to remain readable
- Body text stays comfortably above low-contrast gray values
- Muted text remains readable rather than decorative

## Layout Consistency

- Fixed headers or navbars leave room for page content
- Container widths and spacing rhythm stay consistent
- Visual emphasis follows hierarchy instead of relying on random decoration
