# Planning

## Use This When

- Turning a goal into an execution plan
- Breaking down a feature, migration, or project
- Creating a shared plan before implementation

## Core Workflow

1. Confirm the goal and non-goals.
2. Break work into ordered phases.
3. Note dependencies, verification steps, and open risks.
4. Make the plan portable and tool-agnostic.
