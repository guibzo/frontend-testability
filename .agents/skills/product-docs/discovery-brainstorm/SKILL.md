# Discovery Brainstorm

## Use This When

- A request is under-specified or ambiguous
- You need to uncover constraints, goals, or user needs
- You want structured exploration before planning or building

## Core Workflow

1. Clarify the outcome, audience, and constraints.
2. Surface unknowns and risky assumptions.
3. Generate a small set of credible options.
4. End with concrete decisions or questions that unblock planning.
