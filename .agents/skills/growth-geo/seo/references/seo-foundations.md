# SEO Foundations

Use this reference for practical search visibility work across content quality, site structure, and page experience.

## E-E-A-T Signals

Strong search content demonstrates:

- experience through first-hand examples
- expertise through clear subject knowledge
- authoritativeness through trusted references and recognition
- trustworthiness through accuracy, transparency, and secure delivery

## Technical SEO Basics

Review:

- sitemap and crawl guidance
- canonical handling
- HTTPS coverage
- mobile friendliness
- URL clarity
- schema markup where useful

## Page-Level Content Basics

- title tags are specific and front-load important terms
- meta descriptions describe value clearly
- heading structure is logical
- alt text is descriptive
- content offers original value rather than generic repetition

## Performance Signals

Watch key page experience metrics such as:

- LCP for loading
- INP for interactivity
- CLS for visual stability

## Measurement

Track search performance with ranking, traffic, indexing, and page experience data rather than relying on intuition alone.
