# Mobile UX

## Use This When

- Designing mobile product flows or interfaces
- Adapting a product to touch-first use
- Handling iOS, Android, or cross-platform mobile UX differences

## Core Workflow

1. Start with thumb reach, navigation, and task frequency.
2. Respect platform conventions before styling.
3. Keep performance and readability central to mobile decisions.
4. Move platform-specific guidance into references.
