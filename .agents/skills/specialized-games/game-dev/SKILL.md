# Game Development

## Use This When

- Working on a game or interactive real-time experience
- Deciding between platform-specific or specialty subdomains
- Needing shared guidance on loops, performance, input, or architecture

## Core Workflow

1. Identify target platform and device constraints.
2. Choose the relevant specialty track.
3. Reuse shared gameplay and performance principles across tracks.
4. Keep platform and specialty material in references, not all in the root skill.
